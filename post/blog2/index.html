Java notes：

  • 变量，方法默认权限声明，即所谓的friend友好特性。权限仅小于public，可以被同一个包package中的类调用！

Java方法调用注意事项：

  • 类值为null时，会返回NullPointerException错误，即null.方法名，是非法的！

常用API

1 ArrayList<>() ->数组可变上限（.size()求长度）

（包装类，用于令基本类型假扮成引用类型，用作泛型！）

2 Scanner

3 String

方法：

  • concat(连接)
  • equals
  • index
  • indexOf(首个满足的字符的索引)
  • substring(截取)
  • replace(替换)
  • split(分割符->正则表达式) (分割)

注意：

  • 是常量，不能改变

  • 字符串(引用类型) “==” 比较的是地址

  • String a;
a = "a_c";
System.out.println(a);
a = "b_d";
System.out.println(a);
/*
 a的地址改变了，而地址中的内容没有改变
 而a.concat(String b),a.substring（int x,int y）等方法中a的地址没有变
